BMI Calculator

A simple Body Mass Index (BMI) Calculator built using Python.
This program calculates a person's BMI using their weight and height and classifies it into different health categories such as Underweight, Normal weight, Overweight, and Obese.

Features:
 Takes weight (kg) and height (meters) as input
 Calculates BMI using the standard formula
 Displays BMI value rounded to 2 decimal places
 Shows BMI category
 Handles invalid inputs using exception handling
 Allows users to calculate BMI multiple times

 BMI Formula:
  BMI is calculated using the formula:
  BMI = weight / (height × height)

Where:
 Weight is in kilograms (kg)
 Height is in meters (m)

BMI Categories:
 BMI Range	Category
 Below 18.5	Underweight
 18.5 – 24.9	Normal weight
 25 – 29.9	Overweight
 30 and above	Obese
 
Technologies Used:
 Python
 Basic Input/Output
 Conditional Statements
